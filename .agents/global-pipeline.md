# Pipeline Global de Desenvolvimento

## COMANDO DE PRE-DEPLOY: /pre-deploy
Se o usuário digitar /pre-deploy em qualquer momento, você deve:
1. Executar as 5 skills estratégicas (seo, analytics, microcopy, guardian, touch) silenciosamente, em cascata.
2. Bloquear absolutamente qualquer modificação visual de layout.
3. Ao finalizar, exibir um CHANGELOG DE BLINDAGEM detalhando o que foi injetado nos bastidores, pausar a execução e declarar a aplicação pronta para a Vercel.
