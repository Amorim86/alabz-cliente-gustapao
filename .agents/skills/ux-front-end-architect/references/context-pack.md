# Referência do Skill UX Front-End Architect

Este arquivo concentra os princípios operacionais do skill para manter o `SKILL.md` enxuto.

## Direção

- Priorize composição, ritmo e atmosfera antes de estilos pontuais.
- Use identidade e conteúdo reais como âncora.
- Preserve rastreabilidade entre estrutura, texto e imagem quando houver origem factual.
- Evite respostas mecânicas ou listas sem função.

## O que analisar

- Estrutura do layout
- Ordem de seções
- Navegação
- Tipografia
- Sistema de cor
- Espaçamento e ritmo
- Tratamento de imagem
- Cards e blocos de conteúdo
- Botões e CTAs
- Responsividade
- Acessibilidade
- Riscos e ambiguidades

## Leitura local no Alabz V4

Quando este skill for usado nesta base, os contratos locais de UX continuam sendo a referência principal:

- `docs/ux/CONTEXT_PACK.md`

## Critério de qualidade

Uma boa resposta deixa claro:

- o foco dominante da tela
- onde existe respiro e onde existe acento
- como a hierarquia está distribuída
- como o layout muda no mobile
- o que o implementador precisa fazer sem adivinhar
